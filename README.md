## Description

Calories And Meal is a mobile application that allows you to calculate your daily calorie needs and create personalized meal plans according to your goals.

This project is part of the React Native course taught by [Youcef Acheuk](https://www.linkedin.com/in/youcefacheuk/).

You can find the subject of the project [here](https://react-native-course-esiee.netlify.app/blog/project).

## Technologies used
For this project, we used React Native and we were able to emulate the application on our smartphones thanks to Expo.

[![Expo](https://img.shields.io/badge/Expo-000000?style=for-the-badge&logo=expo&logoColor=white)](https://expo.io/) [![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)


## Guide

### 1. Clone the project

```bash
git clone https://github.com/zkimi/CaloriesAndMeal
```

### 2. Install dependencies

```bash
npm i
```

### 3. Run the app

```bash
npx expo start
```
